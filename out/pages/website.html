This website is pretty slick

This website was a huge amount of fun to make. Here is a few technical details:

  • It's a static website - this is hosted on Netlify.
  • This is really is all in one page. The JavaScript code inserts any relevant HTML directly into the main content area. The pages themselves are only partial and depend on index.html and its styling to display properly. It's pretty fun to watch the animations.
  • There is no frameworks of any kind (if you don't count the HTML5 Boilerplate). There is an automatic polyfill script - but that's obviously just for compatibility.
  • I am practically responsible for all of the site's markup and code (excluding the obvious stuff) - the layout, styling, animations, and most of the JavaScript I created myself.
  • This is my first website. I got initial CSS practice from making modules on the SCP Foundation wiki.
  • Have I stroked my own ego enough? I'm just really happy with how this turned out!
